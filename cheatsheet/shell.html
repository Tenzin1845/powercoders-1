Shell Cheatsheet

  • Machine is a physical computer and hardware (disks, keyboard, etc).
  • Runs an operating system, which manages access to everything.
  • You interact with the OS through the Shell.
  • You use the shell to tell the Operating System which programs to run, it runs them.
  • The shell is just another program.
  • Two types of Shell
    • Graphical
    • Command line

To open Command line

  • Windows: Window key + cmd
  • OSX:Command Key + Space, type terminal
