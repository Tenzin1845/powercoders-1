Cascading style Sheet Cheatsheet

Introducing styling to the web

  • HTML says "What it is", not "What it should look like"
  • Cascading Style Sheets (CSS) say what it should look like

CSS Rule


      selector {
      property_1: value_1;
      property_2: value_2;
      property_3: value_3;
      ...
      }

Link CSS file to HTML File

  • Add link tag to link CSS file to HTML file
  • Syntax:<link rel="stylesheet" type="text/css" href="style.css">

CSS Selectors

Selecting all elements of a particular type


      p {
      color: red;
      background-color: lightgrey;
      margin-left: 1em;
      }

Styling one element using id

  • All elements support the id attribute.
  • Value must be unique in the HTML

      p#id-name {
      color: red;
      background-color: lightgrey;
      margin-left: 1em;
      }

Styling the elements using class

  • All elements support the class attribute.
  • Value does not need to be unique in the HTML

      p.class-name {
      color: red;
      background-color: lightgrey;
      margin-left: 1em;
      }

Styling elements based on container


      p em {
      color: red;
      background-color: lightgrey;
      margin-left: 1em;
      }

Here CSS will be applied to all the <em> element which is nested inside <p> element
